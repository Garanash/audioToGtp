# Рекомендации по оборудованию для ускорения обработки

## Разделение треков (Demucs)

### CPU
- **Рекомендуется**: 8+ ядер (AMD Ryzen 7/9, Intel i7/i9)
- 2-минутный трек на 8-ядерном CPU: примерно 5–12 минут
- 4 ядра: ~15–25 минут на 2 мин трека

### GPU (основное ускорение)
- **Минимум**: NVIDIA GTX 1060 6GB
- **Оптимально**: RTX 3060 12GB / RTX 4070 8GB
- **Максимум**: RTX 4090 24GB

С GPU 2-минутный трек обрабатывается за **~1–3 минуты** вместо 10+ на CPU.

**Включение GPU**: добавьте в `.env` или systemd-сервис:
```bash
DEMUCS_DEVICE=cuda
```

Требуется PyTorch с поддержкой CUDA:
```bash
pip uninstall torch torchaudio -y
pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu121
```

### Память (RAM)
- **Минимум**: 8 GB
- **Рекомендуется**: 16 GB для длинных треков
- Demucs использует ~2–4 GB на задачу

### Celery (параллелизм)
По умолчанию 1 воркер. Для CPU можно поднять до 2:
```ini
ExecStart=... celery -A celery_app worker -l info -c 2
```
На GPU лучше оставить 1 воркер — память видеокарты ограничена.

## MIDI-конвертация и GTP
Требует меньше ресурсов. Достаточно 2–4 ядер и 4–8 GB RAM.
